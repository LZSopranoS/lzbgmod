//V5.4
新增“升级速度调整” 组件
使用方法：确保在所有涉及修改经验的MOD后安装，比如去经验上限补丁；如果不改升级经验则可以随便安装，没有安装顺序要求；兼容40级/50级规则。
组件简介：虽然叫“升级速度调整”组件，其实调整的是每级升级经验，借鉴了兼职经验分配的思路和称呼，分为双职业升级速度和三职业升级速度，双职业升级速度即原职业升级经验*2，三职业升级速度即原职业升级经验*3，安装时提供5种选择。
1、1~10级正常升级速度，11级开始采用双职业升级速度(单职业变双职业、双职业变四职业、三职业变六职业)
2、1级开始就采用双职业升级速度(单职业变双职业、双职业变四职业、三职业变六职业)
3、1~10级正常升级速度，11级开始采用三职业升级速度(单职业变三职业、双职业变六职业、三职业变九职业)
4、1级开始就采用三职业升级速度(单职业变三职业、双职业变六职业、三职业变九职业)
5、1~10级正常升级速度，11~30级双职业升级速度，31级开始采用三职业升级速度(满级所需经验同另外两种三职业升级速度)

3、同时为敌人和玩家开启拟像AI 
4、关闭拟像AI
5、退出

1、效果强化持续4小时，技能释放有概率性，基本是强化前技能越多（喝药水之类的也算）释放职业强化技能的概率越大。

2、如果觉得职业技能强化没起效，请先用控制台命令把生物抓过来确认一下职业，有一些生物会有动画和职业不匹配的情况，比如，烛堡的法师其实不是法师，刺客职业其实是战士。

3、单职业法师/法师类都包含术士，德鲁伊类包含萨满，勇士类包含战士/游侠/圣武士/武僧；比如，职业是单法师的巫妖/半巫妖有幸运加速、施法技能、种族强化；职业是TANARRI的恶魔或职业是DRAGON的龙只有幸运加速、快速治疗/远程/近战攻击特效、种族强化，不包含技能强化。
